# URL Checker

Многопоточная проверка доступности URL по TCP.

Программа создана в учебных целях для практики системного и сетевого программирования на C.

## Возможности

- Парсинг URL с выделением схемы, хоста, порта и пути
- Проверка TCP-соединения с заданным тайм-аутом (поддерживаются оба протокола: http и https)
- Обработка списка URL из файла в несколько потоков (один писатель, несколько читателей)
- Проверка одного URL, переданного аргументом командной строки
- Цветной вывод результата (зелёный — доступен, красный — недоступен)

## Ограничения

- Программа работает только в UNIX-подобных системах
- При проверке файла все URL читаются в память
- Проверка соединения выполняется на уровне TCP

## Сборка

```bash
make
```

## Использование

Проверка одного URL:
```bash
./checker http://example.com
```

Проверка списка URL из файла (по одному на строку):
```bash
./checker -f urls.txt
```

## Как это работает

Один поток-писатель читает и парсит URL из файла, помещая их в кольцевой буфер. Несколько потоков-читателей забирают URL из буфера и проверяют доступность через TCP-соединение с тайм-аутом. Синхронизация выполнена с помощью мьютекса и условных переменных
