# Static Web Server

Простой статический веб-сервер, написанный на C в учебных целях.  
Сервер раздаёт файлы из указанной корневой директории, поддерживает листинг каталогов и корректно обрабатывает ошибки.

## Сборка

```bash
gcc -o server main.c server.c -std=gnu99
```

## Запуск

```bash
./server [port] [root]
```

`port` – номер порта (по умолчанию 8080)

`root` – корневая директория (по умолчанию public/)

Примеры:

```bash
./server                    # порт 8080, корень public/
./server 3000               # порт 3000, корень public/
./server 3000 /var/www      # порт 3000, корень /var/www
```

Сервер слушает на всех интерфейсах, поэтому доступен с любого устройства в локальной сети:

```bash
http://<IP_компьютера>:8080
```

## Возможности

* Отдача статических файлов с определением MIME-типа

* Автоматический листинг каталогов, если нет `index.html`

* Обработка ошибок с информативными HTML-страницами (400, 403, 404, 405, 500)

* Проверка безопасности: запрет выхода за корневую папку и доступа к скрытым файлам

* Логирование IP-адресов клиентов в консоль

## Поддерживаемые типы

| Расширение    | Content-Type             |
| :------------ | :----------------------- |
| .html	        | text/html                |
| .css	        | text/css                 |
| .js	          | application/javascript   |
| .png	        | image/png                |
| .jpg, .jpeg	  | image/jpeg               |
| .gif	        | image/gif                |
| .svg	        | image/svg+xml            |
| .txt	        | text/plain               |
| .pdf	        | application/pdf          |
| другие	      | application/octet-stream |

## Ограничения

* Поддерживается только метод GET

* Сервер однопоточный, обрабатывает запросы последовательно

* Нет поддержки HTTPS

* Нет обработки CGI/скриптов

* Размер буфера листинга ограничен ~16 КБ (при большом количестве файлов возможна обрезка)
