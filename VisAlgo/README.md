# VisAlgo – Визуализация алгоритмов и структур данных

**VisAlgo** — это учебная программа для визуализации классических алгоритмов сортировки, поиска и работы с динамическими структурами данных (стек, очередь). Проект написан на языке C с использованием графической библиотеки **Raylib** и предназначен для демонстрации пошагового выполнения алгоритмов.

## Возможности

### Алгоритмы сортировки (пошаговая анимация)
* Сортировка пузырьком (Bubble sort)
* Шейкерная сортировка (Shaker sort / Cocktail shaker sort)
* Сортировка вставками (Insertion sort)
* Сортировка Шелла (Shell sort)

### Алгоритмы поиска
* Линейный поиск (Linear search)
* Бинарный поиск (Binary search) – требует предварительно отсортированного массива

### Динамические структуры
* **Стек** – горизонтальное отображение, вершина подсвечена красным
* **Очередь** – горизонтальное отображение, голова – красная, хвост – синий

### Управление
* Ручной пошаговый режим (`STEP`)
* Автоматический режим с регулируемой скоростью (`AUTO`, `STOP`, кнопки `+` / `-` для скорости)
* Перемешивание массива (`SHUFFLE`)
* Изменение размера массива (кнопки `<` и `>` рядом с числом элементов)
* Сохранение и загрузка массива в текстовый файл (`SAVE` / `LOAD`)
* Для структур: добавление (`ADD`), удаление (`DELETE`), автоматическое заполнение (`FILL`) и очистка (`RESET`)

### Интерфейс
* Разделение на режимы: работа с массивом (алгоритмы) и работа со структурами
* Меню с переключаемыми группами кнопок (сортировки, поиск, структуры, действия с файлами)
* Нижняя панель с настройками и отображением текущего шага, скорости, количества элементов

## Управление

| Режим         | Кнопка      | Действие                                      |
| :------------ | :---------- | :-------------------------------------------- |
| **Массив**    | `STEP`      | Один шаг выбранного алгоритма                 |
|               | `AUTO`      | Включить/выключить автоматический режим       |
|               | `STOP`      | Остановить авто-режим                         |
|               | `SHUFFLE`   | Перемешать массив                             |
|               | `RESET`     | Сбросить алгоритм (очистить подсветку)        |
|               | `<` / `>`   | Уменьшить/увеличить размер массива            |
|               | `+` / `-`   | Увеличить/уменьшить скорость авто-режима      |
|               | `SAVE`      | Сохранить массив в файл `array.txt`           |
|               | `LOAD`      | Загрузить массив из файла `array.txt`         |
| **Структуры** | `ADD`       | Добавить случайный элемент                    |
|               | `DELETE`    | Удалить элемент                               |
|               | `FILL`      | Автоматически заполнить структуру до ёмкости  |
|               | `RESET`     | Автоматически очистить структуру              |


> **Примечание:** Кнопки управления для структур появляются только после выбора стека или очереди в меню.

## Системные требования

* **Операционная система**: Linux (с X11) или Windows
* **Библиотеки**: Raylib (версия 4.0 или новее)
* **Компилятор**: GCC (MinGW для Windows) с поддержкой стандарта C99

## Сборка и запуск

### На Linux (Ubuntu/Debian)

1.  Установите зависимости и Raylib:
    ```bash
    sudo apt update
    sudo apt install build-essential git cmake libasound2-dev libx11-dev libxrandr-dev libxi-dev libgl1-mesa-dev libglu1-mesa-dev libxcursor-dev libxinerama-dev libwayland-dev libxkbcommon-dev
    ```

2.  Соберите Raylib из исходников:
    ```bash
    git clone --depth 1 https://github.com/raysan5/raylib.git
    cd raylib
    mkdir build && cd build
    cmake -DBUILD_SHARED_LIBS=ON ..
    make -j4
    sudo make install
    sudo ldconfig
    ```

3.  Клонируйте репозиторий и перейдите в папку проекта:
    ```bash
    git clone https://github.com/yourusername/visalgo.git
    cd visalgo
    ```

4.  Скомпилируйте программу:
    ```bash
    gcc -Iinclude src/*.c -o visalgo -lraylib -lGL -lm -lpthread -ldl -lrt -lX11
    ```

5.  Запустите:
    ```bash
    ./visalgo
    ```

### На Windows (MinGW)

1.  Установите [MSYS2](https://www.msys2.org/) или [Git Bash](https://gitforwindows.org/).
2.  Установите Raylib: скачайте с [официального сайта](https://www.raylib.com/) или через `vcpkg`.
3.  Поместите заголовочные файлы Raylib в `C:/raylib/include`, а библиотеки в `C:/raylib/lib` (или укажите свои пути).
4.  В корневой папке проекта выполните:
    ```bash
    make
    ```
    Исполняемый файл появится в `bin/visalgo.exe`.

    Если `make` не установлен, скомпилируйте вручную (пример):
    ```bash
    gcc -Iinclude -IC:/raylib/include src/*.c -o visalgo.exe -LC:/raylib/lib -lraylib -lgdi32 -lwinmm -lopengl32
    ```

## Ограничения

* Размер массива: **4…50** элементов.
* Значения элементов: целые числа от **1 до 100**.
* Бинарный поиск работает только на **отсортированном массиве**; при попытке запуска на неотсортированном выдаётся предупреждение.
* Ёмкость стека и очереди: **10** элементов (задаётся константой `CAPACITY`).
* Сохранение/загрузка осуществляется в файл `array.txt` в текущей директории.
